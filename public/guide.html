이 보고서는 무엇인가요?

한 줄 요약: "우리 회사가 산업 전환(그린, 디지털, AI, 고용구조)에 얼마나 준비되어 있는지를 진단한 결과표"입니다.

왜 이 진단을 하나요?

산업 환경이 빠르게 변하고 있습니다. 탄소중립, 디지털 전환, AI 도입, 고용구조 변화 등 한꺼번에 여러 가지 변화가 닥치고 있습니다. 이런 상황에서 "우리 회사는 어디가 강하고 어디가 약한지"를 객관적으로 파악해야 올바른 대응 전략을 세울 수 있습니다.

G-DAX란?

G-DAX는 4개 영역의 첫 글자를 딴 이름입니다. 각 영역마다 5개 문항, 총 20개 문항으로 귀사의 전환 준비도를 평가합니다.

G Green — 친환경 전환
D Digital — 디지털 전환
A AI — 인공지능 전환
X eXployment — 고용전환

문항 구조: 3단 분류

20개 문항은 각각 아래 3가지 그룹 중 하나로 분류됩니다. 이 분류가 보고서의 핵심 분석 기반입니다.

그룹의미해당 문항 예시
전환 압력외부에서 밀려오는 변화 압력G1, G2, D1, D2, A1, A2, X1, X2 등
실행 기반현재 갖추고 있는 시스템/인프라G4, D3, A3, X3, X4 등
성장 동력 / 인재 역량 / 추진 동력변화를 실행할 수 있는 역량·동력G5, D4, D5, A4, A5, X5 등
점수가 높다고 무조건 좋은 것이 아닙니다!
전환 압력 문항은 점수가 높을수록 "외부 압력이 높다"는 의미이고,
실행 기반·역량 문항은 점수가 높을수록 "내부 준비도가 높다"는 뜻입니다.
즉, "전환 압력은 높은데 준비도가 낮으면" 시급히 대비해야 합니다.

📊 3.2 종합 진단 결과 (전환압력 기반 100점 환산)

보고서에서 가장 먼저 눈에 들어오는 부분입니다. 큰 원 안에 100점 환산 점수가 표시되고, 오른쪽에 4개 영역별 전환 압력 막대 그래프가 나옵니다.

핵심 변경: 종합 점수는 20문항 전체가 아닌, "전환 압력" 그룹에 해당하는 문항만 합산하여 100점으로 환산한 값입니다. 영역별 막대 그래프도 전환 압력 문항의 평균을 표시합니다.

총점 읽는 법

  • 100점 환산 점수 = 전환 압력 문항 합계 ÷ 전환 압력 문항 만점 × 100
  • 총점 아래의 "전환 압력: ○○ (등급)" 텍스트가 전체적인 전환 압력 수준입니다

전환 압력 등급 기준

등급평균 점수의미대응 수준
A등급4.0 ~ 5.0매우 높음즉시 대응이 필요합니다
B등급3.0 ~ 3.9높음체계적인 전환 계획 수립이 필요합니다
C등급2.0 ~ 2.9보통선제적 준비를 시작하기 좋은 시점입니다
D등급1.0 ~ 1.9낮음미래 대비가 필요합니다

영역별 막대 그래프

G(초록), D(파랑), A(보라), X(빨강) 4개 막대가 각각 전환 압력 문항의 평균(5점 만점)을 보여줍니다. 막대 옆에 高/中/低 표시가 함께 나옵니다.

핵심: 4개 막대의 길이 차이를 비교하세요. 특정 영역만 유독 길다면 해당 영역의 전환 압력이 높아 우선 대응이 필요합니다. 이 점수는 전환 압력 문항만 반영하므로, 내부 준비도는 갭 분석(③)에서 확인하세요.
🕸 3.3 G-DAX 전환압력 vs 실행기반·역량 갭 분석

4개 축(G, D, A, X)에 두 개의 다각형이 겹쳐 표시되는 레이더 차트와, 옆에 갭 매트릭스 테이블이 함께 배치된 통합 분석 카드입니다.

이중 다각형 레이더 차트 (왼쪽)

빨간 실선 다각형은 전환 압력 문항 평균을, 파란 점선 다각형은 실행기반+역량+동력 문항의 가중평균을 나타냅니다.

  • 중심(0점)에서 바깥쪽(5점)으로 갈수록 점수가 높습니다
  • 빨간 다각형(전환 압력)이 클수록 외부에서 밀려오는 변화 압력이 강합니다
  • 파란 다각형(실행기반·역량)이 클수록 내부 준비가 잘 되어 있습니다
  • 두 다각형 사이의 분홍 영역이 GAP(갭)입니다 — 이 갭이 클수록 시급한 대응이 필요합니다

마커 범례

◆ 전환 압력 — 전환 압력 문항 평균
● 실행기반·역량 — 실행 기반 + 동력/역량 가중평균
GAP 영역 — 압력과 준비도 사이의 격차

갭 매트릭스 테이블 (오른쪽)

레이더 차트 옆에 배치된 테이블에서 영역별 전환 압력, 내부 준비도, Gap, 판정(긴급/중점관리/모니터링/양호), 우선순위(★)를 한눈에 비교할 수 있습니다.

판정조건의미
긴급압력 高 + 준비도 ≤ 2.5즉시 대응 필수 (★★★)
중점관리압력 高 + 준비도 ≤ 3.5우선적 관리 대상 (★★)
모니터링압력 中지속적 관찰 필요 (★)
양호그 외현 수준 유지 관리 (-)
해석 예시: G(친환경) 축에서 빨간 ◆가 바깥(4.2점)이고 파란 ●이 안쪽(2.1점)이라면?
→ "친환경 전환 압력은 매우 높지만 내부 준비는 미흡한 상태"입니다. 매트릭스 테이블에서 "긴급 ★★★"으로 표시됩니다.
📊 3.4 구조별 소계 분석

20개 문항을 4개 영역(G, D, A, X) × 3단 구조(전환 압력 / 실행 기반 / 동력·역량)로 교차 분석한 매트릭스 테이블입니다.

매트릭스 구조

행 (그룹)G 친환경D 디지털A AIX 고용전환
전환 압력각 영역의 전환 압력 문항 평균 (색상: 빨강 높음 / 파랑 보통 / 녹색 양호)
실행 기반각 영역의 실행 기반 문항 평균
성장 동력 / 인재 역량 / 추진 동력영역별로 다른 3번째 그룹 평균 (G→성장동력, D·A→인재역량, X→추진동력)
핵심: "압력은 높은데 기반과 역량은 낮다" = 가장 위험한 조합입니다.
매트릭스 하단에 자동으로 인사이트가 표시됩니다: "⚠️ ○영역의 전환 압력 대비 실행 기반 갭차가 크므로 우선적 인프라 강화가 필요합니다"와 같은 경고 메시지를 확인하세요.
🔍 3.5 영역별 상세 진단

G, D, A, X 4개 영역 각각에 대해 5개 문항의 개별 점수를 보여줍니다. 각 영역은 다음 요소로 구성됩니다:

  • 영역 헤더: 영역명, 전체 평균, 압력 수준(高/中/低), 준비도 점수
  • 그룹 라벨: 문항들이 "전환 압력", "실행 기반", "성장 동력/인재 역량/추진 동력" 그룹으로 구분되며, 각 그룹의 소계 평균이 표시됩니다
  • 문항별 점수 도트: 1~5점 도트로 각 문항의 점수를 시각화
  • BTJR 태그: 각 문항 옆에 B T 등의 태그가 표시되어, 해당 문항이 BTJR의 어느 경로에 영향을 미치는지 알 수 있습니다

그룹 라벨 색상

그룹색상의미
전환 압력빨간색 계열외부 전환 압력 수준
실행 기반파란색 계열현재 시스템/인프라 수준
성장 동력주황색 계열G 영역의 미래 성장 역량
인재 역량보라색 계열D, A 영역의 인력 역량
추진 동력녹색 계열X 영역의 조직 추진력
핵심: 같은 영역 내에서 전환 압력 그룹 소계는 높은데 실행 기반·역량 그룹 소계는 낮은 패턴을 찾으세요. 이것이 해당 영역에서 가장 시급한 개선 포인트입니다.
🏷 3.6 전환 유형 분류

귀사가 4가지 전환 유형 중 어디에 해당하는지를 보여주는 핵심 카드입니다. 4분면 차트에 "우리 기업" 포인트가 표시됩니다.

분류 로직 (G × D+A 모델):
- X축 = G 전환 압력 문항만의 평균 (환경전환 압력)
- Y축 = D + A 전환 압력 문항만의 가중평균 (기술전환 압력)
- 업종 유형에 따라 D와 A의 가중치가 적용됩니다 (제조형 D55:A45, 서비스형 D50:A50, 기술형 D35:A65)
- 기준선 3.0을 기준으로 4분면에 배치
주의: 전환 유형 분류에는 각 영역의 전체 평균이 아닌, "전환 압력" 그룹에 해당하는 문항만 사용합니다. 또한 X(고용전환) 영역은 유형 분류에 직접 참여하지 않고, 별도의 "고용전환 강도"로 Task Pool의 강도를 조절합니다.

  • 점 아래에 "G압력 ○.○ / D+A압력 ○.○" 좌표값이 표시됩니다
  • 해당 유형이 하이라이트(활성화) 표시됩니다
핵심: 전환 유형은 "어떤 전환과제를 우선 추진할지"를 결정하는 가장 중요한 기준입니다. 아래 Task Pool(⑩번 카드)의 과제 목록이 이 유형을 기반으로 제시됩니다.
💥 3.7 X영역: 고용전환 강도 판정

X(고용전환) 영역의 전환 압력(X1·X2)과 추진동력(X5)만을 사용한 가중평균으로, 인력 구조 변화의 규모를 3단계로 판정합니다.

X영역의 역할: 전환 유형은 G × D+A로 결정되고, X영역은 "전환을 실행할 때 조직이 감당해야 할 변화의 규모"를 판정합니다. 즉, X영역은 Task Pool의 강도 조절자입니다.
계산 방식: 전환 압력(X1·X2) × 가중치 2 + 추진동력(X5) × 가중치 1의 가중평균
→ 실행기반 문항(X3·X4)은 역방향 문항으로 강도 판정에서 제외됩니다. 이는 "외부 압력의 크기"만으로 인력 전환 규모를 판단하기 위함입니다.
강도기준의미보고서 표시
🔥 대규모가중평균 ≥ 3.5인력구조 재편과 노사협력이 필수전환 압력, 추진 동력, 실행 기반 각각의 점수와 강도 점수 표시
⚡ 중규모가중평균 2.5 ~ 3.4단계적 인력 전환과 교육 강화 필요
✅ 소규모가중평균 < 2.5점진적 역량 개발 중심으로 대응

보고서에는 전환 유형 + 고용전환 강도가 결합되어 표시됩니다. 예: "구조전환형 + 대규모 고용전환"

🔗 3.8 BTJR 전환 영향 경로 분석

BTJR은 전환이 기업 내부에 미치는 영향을 4가지 경로로 분석한 것입니다.

경로영문의미
BBusiness Structure사업 모델과 수익 구조에 미치는 영향
TTechnology & Operations생산/운영 방식에 미치는 영향
JJob & Competency필요 직무와 역량에 미치는 영향
RLabor Relations고용 관계와 소통에 미치는 영향

BTJR 연쇄 흐름

보고서 상단에 B → T → J → R 순서로 연결된 흐름도가 표시됩니다. 사업구조(B)가 바뀌면 → 기술운영(T)이 전환되고 → 직무역량(J)이 재편되며 → 노사관계(R)의 뒷받침이 필수라는 연쇄 관계를 보여줍니다.

경로별 상세 카드 (신규)

각 BTJR 경로에 대해 다음과 같은 상세 분석 카드가 제공됩니다:

  • 갭 바(Gap Bar): 압력·동력 점수와 실행·역량 점수를 막대로 비교하고, Gap 값과 함께 ⚠긴급 / 🟡주의 / ✅양호 판정이 표시됩니다
  • 압력·동력 테이블: 해당 경로에 기여하는 전환 압력·성장 동력·추진 동력 문항들의 상세 목록 (문항 코드, 항목명, 그룹, 점수)
  • 실행·역량 테이블: 해당 경로에 기여하는 실행 기반·인재 역량·성장 동력·추진 동력 문항들의 상세 목록
BTJR 점수 산출 방식: 각 문항은 BTJR 경로에 1/n 균등 분배 방식으로 가중 매핑됩니다. 예를 들어 문항이 [B, T] 두 경로에 걸쳐 있으면 각각 1/2 가중치로 배분됩니다.
핵심: 갭 바에서 "Gap이 1.0 이상"인 경로가 가장 취약한 부분입니다. 해당 경로의 상세 테이블에서 어떤 문항들이 기여하는지 확인하면, 구체적인 개선 포인트를 찾을 수 있습니다.
🧩 3.9 Cross-SWOT 예비 분석

G-DAX 3단 구조(전환 압력 × 실행기반·역량 갭 분석)를 기반으로 한 예비 SWOT 분석입니다.

예비 분석이란? 이 SWOT은 사전진단 응답 기반의 예비 분석입니다. 정밀 SWOT은 정밀 진단(FABRIC 기반) 이후 완성됩니다.
항목의미기준
S (강점)이미 잘하고 있는 것내부 준비도(실행기반·역량) 문항 4점 이상
W (약점)보완이 필요한 것내부 준비도 문항 2점 이하
O (기회)선제 대응 가능한 기회준비도 높음 + 전환 압력 높음 (선제 대응 가능)
T (위협)시급히 대비해야 할 위협전환 압력 높음 + 준비도 낮음
핵심: SWOT에서 가장 중요한 것은 W(약점)과 T(위협)의 교집합입니다.
"내부 역량도 부족하고 외부 위협도 큰 영역"이 최우선 개선 대상입니다.
📋 3.10 전환 유형별 전환과제 추천 (Task Pool)

귀사의 전환 유형에 맞는 구체적인 전환과제를 추천하는 카드입니다. 유형별로 표준 Task Pool이 정의되어 있으며, 3~4개의 전환 방향과 세부 과제가 제시됩니다.

Task Pool 구성

  • 전환 방향: 큰 틀에서의 추진 방향 (예: "친환경·디지털 이중전환")
  • 세부 과제: 구체적으로 실행할 과제 (예: "친환경 운영 체계 전환")
  • G-DAX 연계: 해당 과제와 연관된 진단 문항 (예: "G1~G5")
  • 과제 설명: 무엇을 어떻게 할 것인지 상세 설명

고용전환 강도에 따른 과제 조정 (xScale)

일부 과제(특히 인력 전환·노사관계 관련)는 고용전환 강도에 따라 과제명·설명·방향이 동적으로 변경됩니다. 전환 방향 제목 옆에 강도 배지(🔥 대규모 / ⚡ 중규모 / ✅ 소규모)가 표시됩니다.

강도과제 조정 방향예시
🔥 대규모대규모 인력재편·노사합의 중심"직무 재설계 및 전환배치" → 구체적 재편 계획
⚡ 중규모단계적 전환·교육훈련 강화"단계적 직무 재설계" → 점진적 역량 전환
✅ 소규모역량 고도화·선제적 준비"직무 역량 진단 및 개발" → 미래 대비 중심

동적 배지(Badge) 이해하기

일부 과제 옆에 색깔 배지가 표시될 수 있습니다:

🚨 시급 관련 영역 점수 2.0 미만 — 즉시 대응 필요
⚠️ 우선개선 관련 영역 점수 3.0 미만 — 빠른 시일 내 개선 권장
포인트: 같은 전환 유형이라도 고용전환 강도에 따라 보고서에 표시되는 과제 내용이 달라집니다. 이는 기업의 실제 인력 전환 규모에 맞춤화된 과제를 제시하기 위함입니다.
💡 3.11 맞춤형 추천 사항 (우선순위)

보고서의 최종 분석 카드입니다. "압력 × 갭차 × 역량" 복합 스코어링을 기반으로 산출된 우선순위가 표시됩니다.

우선순위 스코어링 공식

각 영역별로 다음 공식으로 복합 점수를 산출하여 높은 순서대로 정렬합니다:

복합 스코어 = 압력강도 × (1 + 압력-기반 갭 × 0.5) × (1 + (5-역량)/4 × 0.3)
→ 전환 압력이 높고, 기반과의 갭이 크며, 역량이 부족할수록 우선순위 점수가 높아집니다.

추천 내용 구성

  • 영역별 전환과제: 압력 3.5 이상인 영역에 대한 구체적 전환 전략
  • Gap 해소 과제: 압력 대비 준비도 격차가 큰 영역의 우선 대응
  • BTJR 연쇄 영향: 사업구조 전환 압력이 높을 때 B→T→J→R 통합 전환 전략
  • 전환 유형 맞춤 컨설팅: 정밀 진단을 통한 기업 맞춤 전환과제 설계 안내

각 추천 항목에는 우선점수와 함께 관련 영역/경로 태그(G, D, A, X, B, T, J, R)가 표시됩니다.

🎯 3.12 희망 지원 분야

설문 시 선택한 정부지원 희망 분야가 표시됩니다. 사업재편, 교육훈련, 스마트공장 등 기업이 원하는 지원 분야를 컨설턴트에게 전달하기 위한 참고 정보입니다.

활용 포인트: 진단 결과에서 도출된 전환 유형 및 추천 과제와, 기업이 직접 선택한 희망 지원 분야를 매칭하여 컨설턴트 미팅 시 논의하면 효과적입니다.

점수 해석 기준

4.1 5점 리커트 척도 해석

모든 문항은 1~5점 척도로 평가됩니다. 점수의 의미는 문항이 속한 그룹에 따라 다릅니다.

전환 압력 문항

5점 = 전환 압력이 매우 높음
4점 = 전환 압력이 높음
3점 = 중간 수준
2점 = 전환 압력이 낮음
1점 = 거의 없음

실행 기반 · 역량 · 동력 문항

5점 = 내부 준비가 매우 잘 되어 있음
4점 = 잘 되어 있음
3점 = 기본 준비 됨
2점 = 준비 부족
1점 = 전혀 준비 안 됨

4.2 영역별 평균 해석

점수 범위수준의미컨설턴트 코멘트
4.0 ~ 5.0매우 높음전환 압력/준비도 최상위적극 대응 → 즉시 실행 계획 필요
3.0 ~ 3.9높음적극적 대응 필요 구간체계적 계획 수립 → 점진적 실행
2.0 ~ 2.9보통선제적 준비 권장 구간중장기 계획 수립 → 역량 강화
1.0 ~ 1.9낮음현재 안정적이나 미래 대비모니터링 → 변화 감지 시 대응

4.3 高/中/低 표시 기준

표시기준의미
高평균 ≥ 3.5높은 전환 압력 — 적극 대응 필요
中평균 2.5 ~ 3.4중간 수준 — 모니터링 및 준비
低평균 < 2.5낮은 전환 압력 — 선제적 대비

자주 묻는 질문(FAQ)

Q1 "점수가 높으면 좋은 건가요, 나쁜 건가요?"
문항이 속한 그룹에 따라 다릅니다. 전환 압력 문항에서 높은 점수는 "외부 압력이 높다"는 뜻이므로 빨리 대응해야 한다는 신호입니다. 실행 기반·역량 문항에서 높은 점수는 "내부 준비가 잘 되어 있다"는 긍정적 의미입니다. 결국 중요한 것은 "압력과 준비도 사이의 갭(차이)"입니다.
Q2 "종합 점수가 20문항 전체 합산이 아니라고요?"
맞습니다. 종합 점수(100점 환산)는 전환 압력 그룹에 해당하는 문항만 합산하여 100점으로 환산한 값입니다. 실행 기반이나 역량 문항은 포함되지 않습니다. 이는 "우리 회사가 직면한 외부 전환 압력의 크기"를 명확히 보여주기 위한 설계입니다. 내부 준비도는 갭 분석(③번 카드)에서 별도로 확인할 수 있습니다.
Q3 "전환 유형이 결정되면 바꿀 수 없나요?"
전환 유형은 현재 시점의 진단 결과입니다. 산업 환경이 변하거나 기업이 전환 노력을 하면 유형은 바뀔 수 있습니다. 예를 들어, "강소기반형"이었던 기업이 환경 규제가 강화되면 "가치창출형"으로 변할 수 있습니다. 정기적인 재진단(6개월~1년 주기)을 권장합니다.
Q4 "추천된 과제를 모두 다 해야 하나요?"
모든 과제를 동시에 추진할 필요는 없습니다. 맞춤형 추천 사항(⑪번 카드)의 우선순위 분석을 참고하여 가장 시급한 1~2개 과제부터 시작하는 것을 권장합니다. 특히 🚨 시급 배지가 붙은 과제가 있다면 해당 과제를 최우선으로 검토하세요.
Q5 "이 보고서 결과로 정부 지원을 받을 수 있나요?"
네, 가능합니다. G-DAX 진단 결과와 전환 유형은 고용노동부·산업통상자원부 등의 산업전환 지원사업 신청 시 근거 자료로 활용할 수 있습니다. 보고서 하단의 희망 지원 분야(⑫번 카드)에서 선택한 분야와 매칭하여, 담당 컨설턴트와 적합한 정부지원 프로그램을 논의하시기 바랍니다.
Q6 "보고서에서 가장 먼저 봐야 할 부분은?"
시간이 제한적이라면 다음 3개만 보세요:
(1) 종합 진단 결과(②)의 100점 환산 점수와 등급
(2) 전환 유형 분류(⑥) — 우리 회사가 4가지 중 어디인지
(3) 전환과제 추천(⑩)의 Task Pool
이 3가지만으로도 "우리 회사의 현재 위치와 해야 할 일"을 충분히 파악할 수 있습니다.

📥 6.4 PDF 다운로드
  • 보고서 하단의 "📥 PDF 다운로드 (A4)" 버튼을 클릭하면 브라우저의 인쇄 기능을 통해 PDF로 저장할 수 있습니다
  • PDF 파일명은 자동으로 "GDAX_준비도진단_결과_(회사명)_(날짜)" 형식으로 설정됩니다
  • 차트(레이더, 4분면)도 이미지로 변환되어 PDF에 포함됩니다
